# AnkenList 設計書

## 1. 概要
案件一覧ページのメインコンテンツ部分を担うコンポーネント。
アプリケーション共通の`Layout`コンポーネント（ヘッダー等を含む）の中で使用される。

## 2. 実装タスクリスト
・ [x] ヘッダーの作成 (→ `Header.js`に分離)
・ [x] ロゴの作成とスタイリング (→ `Header.js`に分離)
・ [x] リファクタリング: `AnkenList`から`Header`コンポーネントを分離し、再利用性を高める
・ [ ] 案件一覧テーブルの表示
・ [ ] データのフィルタリング機能

## 3. Props (コンポーネントが受け取るデータ)
・ `items` (Array): テーブルに表示する案件データのリスト。
  ・ `id` (number): 案件ID
  ・ `name` (string): 案件名
  ・ `status` (string): ステータス（例：「進行中」、「完了」）

*(注意: この`items` propは、次のテーブル実装時に使われる予定です)*
